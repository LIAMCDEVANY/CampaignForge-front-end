# CampaignForge

CampaignForge is a MERN stack application for managing tabletop RPG campaigns and session notes.

## Live App
https://your-netlify-url.netlify.app

## Backend API
https://campaignforge-back-end.onrender.com

## Technologies Used
- MongoDB
- Express
- React
- Node.js
- JWT Authentication
- Vite

## Features!

- User authentication
- Campaign management
- CRUD functionality
- Secure API routes

## Next Steps
- Session note editor
- Player management
- Dice rolling tools
